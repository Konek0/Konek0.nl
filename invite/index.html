How to invite Konek0

Konek0 is a private bot, meaning it's not available for everyone to add. If you wish to add Konek0, your server must fit these criteria:

- A minimum of 300 members.
- Not have bots with features that Konek0 already offers.
- An active server with the intention to use the bot.
- Must not abuse the bot.


Konek0 requires some basic JSON knowledge to customize. Please keep that in mind.
If your server meets these requirements, please join the support server and contact VAC Efron#0001 to add Konek0 to your server.
