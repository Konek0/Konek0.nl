How to set up reaction roles

This guide will cover how to set up reaction roles using the interactive setup provided by Konek0.

1) Prepare your message

1. Send a message to add the reaction role to.


2. Enable developer mode and copy the message ID.

2) Create the reaction role

1. Type '.rr create' to start the interactive setup.
2. Type in which channel the message is located.
3. Paste the message ID from earlier.
4. Select what type of reaction role you want (4 options).
5. Type the emoji you want for the reaction role.
6. Type which role the reaction role will give.
