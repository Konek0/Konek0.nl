Your settings file: basic guide

This guide will show you the basics of how to edit your settings file to customize Konek0 for your server.
It will cover the basics of JSON and how to use it.

Variables

Below is a table of the different variables you'll use to customize Konek0.
For example, you'll use 'true' and 'false' to turn things on and off, and you'll replace 'ROLE_ID' with a role's ID of your choice.

Variable Description
true | false Toggles the setting on or off.
ROLE_ID The ID of a role of your choice.
CHANNEL_ID The ID of a channel of your choice.
none | mute | kick | ban The action the bot applies to the user when x happens.

1) Get your settings file

1. Type '.settings' to get your settings file. Konek0 will send a message back with a .json file attached.
2. Download the file to your computer.
3. Open it in any text editor. (We'll use Notepad here)

3) Updating your settings

When you're done editing your settings file, you must upload it by giving it back to Konek0.

1. Save the changes in your file.
2. Go back to Discord and type '.settings upload' with your edited file attached.
3. Done! Your customized settings are now applied.
